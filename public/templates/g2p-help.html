The workbench is designed for seamless visual interaction across various omics data sets within KBase. GWAS experiments comprise observations of variation (typically, SNPs) for a set of studied traits. To select an interesting trait, use the dropdown menus to select a Genome, then an associated Experiment, and finally a Trait of interest. Once you've selected a trait, a Manhattan plot is drawn. The Manhattan plot displays the distribution of SNP variations at varying significance (Y-axis) across the entire genome (X-axis).

Drag a box around a set of highly significant variations to select genes in close proximity (currently within 10,000 bp around each selected variation). The workbench then fetches data associated with those genes:

  • Internal Network: Network-based associations among the selected genes.
  • Expression Profiles: Average expression values for the selected genes across several tissue types
  • Trait Genes: A summary table of the selected genes, along with their annotated function and gene ontology (GO) terms.
  • GO Enrichment: A set of highly-enriched gene functions that are common to the set of genes.

For any of the available widgets, you can export the underlying data by clicking on the gear and selecting "Export data". You can also rearrange the subviews by dragging them with the arrow handle at the top right.
